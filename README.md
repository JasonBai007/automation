# Profile
> Target：借助 npm 和 gulp 实现前端开发自动化.  
> Techs：npm + gulp

### Features
* 自动实现：LESS > CSS > autoprefix > Min > Concat
* 自动实现：JS > Min > Concat
* 随时监控文件的变化
* 浏览器自动刷新
* 功能拓展性强
* 配置文件自由定制
* 自带静态服务器

### Usage
> 1、安装依赖：$ npm install  
> 2、静态文件：$ npm run build  
> 3、自动监控：$ npm start
